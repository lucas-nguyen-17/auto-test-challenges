# Image Helper
### Downloader
Download a bunch of sample images from placeholder-image websites:
- https://placeholder.com/
- https://picsum.photos/
- https://placeimg.com/
- https://place-hold.it/
- http://lorempixel.com/
...

### Compression
- Upload a zip file of images
- Compress these images and return a new zip file of compressed images
